# Flappy Bird с прокачкой (PyGame)

Демо‑проект: меню → игра → пауза → магазин (скины/апгрейды) → сохранение в `save.json`.

## Запуск

1) Установи Python 3.10+  
2) Установи зависимости:

```bash
pip install -r requirements.txt
```

3) Запусти игру:

```bash
python -m src.main
```

## Управление

- **Пробел** — прыжок
- **Esc** — пауза
- **Мышь** — кнопки/магазин

## Картинки (опционально)

Если положить PNG‑картинки в папку `assets/` с нужными названиями — игра автоматически подхватит их.
Если картинок нет — игра всё равно работает (используются аккуратные плейсхолдеры).

Список нужных ассетов и размеров: `assets/ASSETS_LIST.md`.
